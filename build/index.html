Getting started

First, download Prime Grid using download button above or

Clone repository directly to your project

git clone https://github.com/nmihalyov/prime-grid.git

Install via NPM

npm i prime-grid

Install via Yarn

yarn add prime-grid
Usage

If you want to use grid as it is and dont't want to change any variables, you can just put css file into your project folder and import it

<link rel="stylesheet" href="path/to/prime-grid.min.css">

If you installed Prime Grid via package manager and you are using Sass/Scss you can import corresponding file

@import 'node_modules/prime-grid/dist/prime-grid.scss';
@import 'node_modules/prime-grid/dist/prime-grid.sass'

To define your own grid settings you can override variables values before import

/* Defines spacing between columns and rows (default is 16px) */
$gutter: 20px;

/* Dimensions below which new grid styles are applied */
$breakpoints: ("xl": 1600px, "lg": 1280px, "md": 992px, "sm": 768px, "xs": 576px);

/* Sets new max container width below each breakpoint (see $breakpoints) */
$container-max-widths: ("default": 1600px, "xl": 1440px, "lg": 1200px, "md": 960px, "sm": 720px, "xs": 540px);

@import 'node_modules/prime-grid/dist/prime-grid.scss';
/* Defines spacing between columns and rows (default is 16px) */
$gutter: 20px

/* Dimensions below which new grid styles are applied */

/* Sets new max container width below each breakpoint (see $breakpoints) */

Breakpoints

Breakpoints are customizable widths that determine how your responsive layout behaves across devices. Prime Grid has 5 default breakpoints which can be customized via variable override (see Usage)

Breakpoint Class infix Dimensions
Default – All
Extra large xl < 1400px
Large lg < 1200px
Medium md < 992px
Small sm < 768px
Extra small xs < 576px
Container

.container is the main block of layout that contains all other grid elements. You can use as much containers as you want and nest them.

<div class="container">
 ...
</div>

Default containers has limited max-width which can be changed (see Usage). Default maximum widths for containers are:

Breakpoint Class infix Maximum container width
Default – 1320px
Extra large xl 1320px
Large lg 1140px
Medium md 960px
Small sm 720px
Extra small xs 540px

If you want the container to be full width use .container-fluid instead.

<div class="container-fluid">
 ...
</div>

Full and limited container width can be changed on different breakpoints by combining classes: .container-{breakpoint} and .container-fluid-{breakpoint}

<div class="container container-fluid-lg container-sm">
 ...
</div>

Column

.col is a cell for your content that is placed within the .row.

<div class="container">
 <div class="row">
   <div class="col">
     ...
   </div>
   <div class="col">
     ...
   </div>
 </div>
</div>

By default .col will be stretched to fill 100% of the .row. If you place several columns into row they all will be the same widths.

Responsiveness

Prime Grid provides five levels of predefined classes for building complex responsive layouts. You can change your layout behavior depending on your device width (see Breakpoints) using infixes.

.col-{breakpoint} (.col-{breakpoint}-*)

In the example below we will get 2 columns which are 8/12 and 4/12 width by default, 7/12 and 5/12 width on large and below devices, same (half) width on small devices.

<div class="container">
 <div class="row">
   <div class="col-8 col-lg-7 col-sm">...</div>
   <div class="col-4 col-lg-5 col-sm">...</div>
 </div>
</div>
.col-8.col-lg-7.col-sm
.col-4.col-lg-5.col-sm

 </div>

Show and hide any type of contents: containers, row, columns.

   <div class="col">...</div>

 </div>
</div>
Gutter

There is a default horizontal gutter applied to rows and columns. The default gutter value is 16px which can be changed by overriding $gutter value (see Usage).

To add vertical paddings use this class on row or column.

.vert-gutter (.vert-gutter-{breakpoint})

<div class="container">
 <div class="row vert-gutter">
   <div class="col vert-gutter-md">...</div>
   <div class="col vert-gutter-md">...</div>
 </div>
</div>

To disable vertical paddings use this class.

.no-vert-gutter (.no-vert-gutter-{breakpoint})

<div class="container">
 <div class="row vert-gutter no-vert-gutter-md">
   <div class="col">...</div>
 </div>
</div>

To add horizontal paddings use this class on row or column.

.hor-gutter (.hor-gutter-{breakpoint})

<div class="container">
 <div class="row hor-gutter">
   <div class="col hor-gutter-md">...</div>
   <div class="col hor-gutter-md">...</div>
 </div>
</div>

To disable horizontal paddings use this class.

.no-hor-gutter (.no-hor-gutter-{breakpoint})

<div class="container">
 <div class="row no-hor-gutter">
   <div class="col">...</div>
 </div>
</div>

To disable all paddings you can apply special class to row/column.

.no-gutter (.no-gutter-{breakpoint})

<div class="container">
 <div class="row no-gutter">
   <div class="col no-gutter-md">...</div>
   <div class="col no-gutter-md">...</div>
 </div>
</div>

If you want to add both vertical and horizontal paddings just use

.gutter (.gutter-{breakpoint})

<div class="container">
 <div class="row">
   <div class="col no-gutter gutter-md">...</div>
   <div class="col no-gutter gutter-md">...</div>
 </div>
</div>
